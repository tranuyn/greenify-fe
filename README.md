# 🌿 Greenift Mobile App

> Một ứng dụng di động thân thiện, hiện đại được phát triển trên nền tảng **[React Native](https://reactnative.dev/)**.

Greenifile mang đến trải nghiệm mượt mà, giao diện tối ưu và hiệu năng cao cho cả hai hệ điều hành Android và iOS. Đây Là một nền tảng khuyến khích người dùng xây dựng lối sống xanh thông qua các hoạt động thiết thực như đăng hành động xanh, tham gia sự kiện môi trường, tích điểm, đổi quà và theo dõi hành trình phát triển của bản thân. Ứng dụng không chỉ ghi nhận đóng góp cá nhân mà còn tạo ra một cộng đồng cùng nhau lan tỏa các giá trị bền vững.

---

## ✨ Tính năng nổi bật

- 📱 **Cross-platform:** Hỗ trợ hoàn hảo trên cả iOS và Android chỉ với một codebase.
- ⚡ **Hiệu năng cao:** Tối ưu hóa render và quản lý state mượt mà.
- 🎨 **Giao diện hiện đại:** Thiết kế UI/UX thân thiện, chuẩn Material Design / Human Interface Guidelines.
- 🔒 **Bảo mật:** Quản lý xác thực và dữ liệu an toàn.

## 🚀 Getting Started (Hướng dẫn cài đặt)

Để chạy dự án này trên máy cá nhân của bạn, vui lòng làm theo các bước dưới đây.

### 📋 Yêu cầu hệ thống (Prerequisites)

Trước khi bắt đầu, hãy đảm bảo máy tính của bạn đã cài đặt sẵn các công cụ sau:

- [Node.js](https://nodejs.org/) (Khuyên dùng phiên bản LTS mới nhất)
- Trình quản lý gói: `npm` hoặc `yarn`
- Môi trường phát triển: **Visual Studio Code** hoặc **Xcode** (cho máy Mac nếu muốn build iOS).
- Thiết bị di động có cài đặt **Expo go**

### 🛠️ Cài đặt dự án

**Bước 1:** Clone kho lưu trữ này về máy của bạn:

```bash
git clone https://github.com/tranuyn/greenify-fe.git
cd greenify-fe
```

**Bước 2:** Cài đặt các gói phụ thuộc (Dependencies):

```bash
npm install
# or
yarn install
```

**Sau khi cài đặt xong, bạn có thể khởi động Metro Bundler và chạy ứng dụng trên máy ảo (Emulator/Simulator) hoặc thiết bị thật.**
